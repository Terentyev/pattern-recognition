Вариант 5.
В круге радиуса R проводятся хорды параллельно заданному направлению.
Какова вероятность того, что длина наугад взятой хорды не более R, если
равновозможны любые положения точек пересечения хорды с диаметром,
перпендикулярным выбранному направлению?

Примечание:
Для моделирования событий с бесконечным числом исходов, интерпретируемых как
случайный равновозможный выбор точки из некоторого вещественного диапазона
[a,b], включить и использовать следующую функцию
	double drandom(double a, double b) { return a+(b-a)*rand()/RAND_MAX; }
Здесь  rand() - стандартная функция языка С++ (возвращает целое число из
диапазона [0, RAND_MAX]).

Аналитическое решение:
Надо рисовать =)
P = l / r = (r - a) / r = (r - r * sin(pi/3)) / r = 1 - sin(pi/3) =
  = 1 - sqrt(3) / 2 ~= 0,13397459
