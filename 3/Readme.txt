Лабораторная работа  № 3

Тема: Системы случайные величин (случайные векторы) с нормальным законом
распределения  :  отображение двумерных проекций плотности распределения в
изолиниях, моделирование случайных выборок с заданными параметрами
распределения, оценивание параметров закона распределения по выборочным данным

Задание: Каждый студент должен написать программу, позволяющую пользователю
выполнять следующие действия.

1. Ввод параметров нормального закона распределения с консоли либо из файла
	параметров:
M – число случайных переменных в системе (размерность случайного вектора )
 - вектор средних значений
B – корреляционная матрица 
(предпочтителен вариант с заменой ввода матрицы B вводом нормированной
	корреляционной матрицей и вектором дисперсий )

2. Выбор произвольной пары компонентов случайного вектора и отображение
	соответствующего плоской проекции плотности распределения в виде системы
	изолиний.
(область отображения квадратная, геометрические масштабы при отображении
	определяются автоматически и (или) пользователем)

3. Моделирование случайной выборки объема N векторов  с заданным в пункте 1
	законом распределения

4. Сохранение модельной выборки в файл данных в виде таблицы, каждая строка
	которой содержит значения компонент соответствующего случайного вектора из
	выборки

5. Чтение случайных выборок (модельных и реальных) из файлов данных табличного
	вида

6. Отображение точек выборки для произвольных пар компонент случайного вектора
	на соответствующих плоскостях (для модельных данных совместно с изолиниями
	заданной плотности распределения)

7. Расчет и отображение оценок параметров закона распределения  и (либо  и ) по
	имеющейся выборке

8. Отображение оценки плотности распределения в виде системы изолиний для
	произвольных плоских сечений совместно с точками случайной выборки и
	истинной плотностью распределения (если выборка модельная).

В итоговом отчете должны быть проиллюстрированы все возможности написанной программы

Варианты: Без вариантов
